# Vikings Sass

Our goal in this project is to create a web page using Sass.

```

              |----main.css

                    |----_responsive.scss 
              |----pages (folder)

                    |----_contact.scss 
                    |----_home.scss 
                    |----_projects.scss 
              |----main.scss  
        |----img (folder)
```
